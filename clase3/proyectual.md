# Proyectual

> Esta es una pequeña guía acerca de cómo encarar un proyecto para desarrollar un sitio o aplicación basada en la web

1. Definir tema
2. Inverstigación
3. Secciones (barra de navegación)
4. Bocetado (papel)
5. Prototipado (mockup) 
6. Maquetado  
7. Publicación
